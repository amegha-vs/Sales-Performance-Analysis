# 📊 Sales Performance Analysis

## 📌 Project Overview
This project analyzes retail sales data to uncover key business insights and trends. The analysis includes data cleaning, exploratory data analysis (EDA), and an interactive dashboard built using Power BI.

---

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Power BI

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Orders, Average Order Value)
- Sales by Region
- Sales by Category
- Sales Trend Over Time
- Top 5 Products by Sales
- Interactive Filters (Slicers)

---

## 🧠 Key Insights
- Sales performance varies across regions.
- A few product categories dominate total revenue.
- Sales trends indicate seasonal fluctuations.
- Top products contribute significantly to total sales.

---

## 💼 Business Recommendations
- Focus on high-performing regions to maximize revenue.
- Improve strategies in low-performing regions.
- Expand high-demand product categories.
- Promote top-selling products.
